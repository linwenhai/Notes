## 数据类型



### 1 数值类型

| 类型       | 描述                             |
| ---------- | -------------------------------- |
| int(M)     | 整型，默认宽度11                 |
| float(M,D) | 浮点型，M为位数，D为小数保留位数 |



### 2 日期时间类型

| 类型      | 描述         | 例子                |
| --------- | ------------ | ------------------- |
| date      | 年月日       |                     |
| time      | 时分秒       |                     |
| datetime  | 年月日时分秒 |                     |
| year      | 年，默认四位 |                     |
| timestamp | 当前系统时间 | YYYY-MM-DD HH:MM:SS |





### 3 字符串类型

| 类型       | 描述           | 例子 |
| ---------- | -------------- | ---- |
| char(M)    | 固定长度字符串 |      |
| varchar(M) | 可变长度字符串 |      |



### 4 比较运算符

- between
- in
- is null
- is not null
- like
