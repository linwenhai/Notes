## 日志和字符集

#### 1 错误日志

[mysqld]
Log_error=DIR/[filename]

```mysql
日志路径：
mysql> show global variables like 'log_error%';
+---------------------+-----------------------------+
| Variable_name       | Value                       |
+---------------------+-----------------------------+
| log_error           | /app/mysql/data/ceshi01.err |
| log_error_verbosity | 3                           |
+---------------------+-----------------------------+
```

#### 3 慢查询日志

```mysql
mysql> show global variables like 'slow_query%';	#慢查询日志设置
+---------------------+----------------------------------+
| Variable_name       | Value                            |
+---------------------+----------------------------------+
| slow_query_log      | OFF                              |
| slow_query_log_file | /app/mysql/data/ceshi01-slow.log |
+---------------------+----------------------------------+

mysql> show global variables like 'long_query%';	#慢查询时间设置
+-----------------+-----------+
| Variable_name   | Value     |
+-----------------+-----------+
| long_query_time | 10.000000 |
+-----------------+-----------+

mysql> show global variables like 'log_output%';	#日志输出方式
+---------------+-------+
| Variable_name | Value |
+---------------+-------+
| log_output    | FILE  |
+---------------+-------+
```
