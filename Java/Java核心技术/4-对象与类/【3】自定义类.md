## 自定义类



在一个源文件中， 只能有一个公有类，但可以有任意数目的非公有类。  

公有类名与源文件名一致。

关键字 this 表示隐式参数。  

类的方法可以访问本类的任何一个对象的私有域。  



#### 一、创建类

【语法】：

```java
修饰符 class 类名 {
    属性声明;
    方法声明;  
}
```

【示例】：

```java
public class Test {
    String name;    //属性声明
    int age;

    public void study() {    //方法声明
        System.out.println("学习中");
    }
}
```





#### 二、创建对象

类通过new创建对象。

【语法】：

```java
类名 变量名=new 类构造方法;
```
