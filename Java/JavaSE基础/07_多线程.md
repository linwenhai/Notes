



| 方法            | 功能                                   |
| --------------- | -------------------------------------- |
| isAlive()       | 判断线程是否活动（即线程是否还未终止） |
| getPriority()   | 获取线程的优先级                       |
| setPriority()   | 设置线程的优先级                       |
| setName()       | 获取线程的名字                         |
| currentThread() | 获取当前正在运行的线程对象             |
