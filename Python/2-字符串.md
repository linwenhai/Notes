## 字符串

### 1 创建字符串

可以通过单引号或双引号创建字符串。例如：a=’abc’; b=”sxt”。



### 2 常用函数

| 函数         |                                                          |
| ------------ | -------------------------------------------------------- |
| ord()        | 把字符转换成对应的Unicode码                              |
| chr()        | 把十进制数字转换成对应的字符                             |
| len()        | 用于计算字符串含有多少字符                               |
| input()      | 从控制台读取键盘输入的内容                               |
| str()        | 将其他数据类型转换为字符串                               |
| replace()    | 字符串替换                                               |
| \            | 转义字符                                                 |
| end          | 参数end = “任意字符串”，实现末尾添加任何内容             |
| []           | 切片，格式：[起始偏移量：终止偏移量：步长]（包头不包尾） |
| split()      | 分隔字符串，存储到列表中，默认使用空白字符               |
| join()       | 合并字符串                                               |
| strip()      | 去除字符串首尾指定信息，默认去除空格                     |
| capitalize() | 首字母转大写                                             |
| title()      | 每个单词都首字母大写                                     |
| upper()      | 全部字母转大写                                           |
| lower()      | 全部字母转小写                                           |
| format()     | 实现对字符串的格式化                                     |
| encode()     | str 转 bytes                                             |
| decode()     | bytes 转 str                                             |

```python
>>> ord('a')
```


#### 2.8 split()函数


```python

```

#### 2.10 strip()函数

```python
>>> a=" duoduo "
>>> a.strip()
'duoduo'
>>> a.lstrip()

>>> a.rstrip()
' duoduo'
```

#### 2.11capitalize() 、title() 、upper()和lower()函数

```python
>>> a='lin duoduo'

>>> b='Lin duoduo'
>>> b.title()
'Lin Duoduo'
```

#### 2.12 format()函数

```python
>>> a='Myname is {0}'
>>> a.format('duoduo')
'Myname is duoduo'
```

#### 2.13 encode()与decode()函数

```python
>>> a='duoduo'
>>> a.encode()
b'duoduo'
```

```python
>>> b=b'duoduo'

>>> b.decode()

```

### 3 数字格式化

浮点数通过 f，整数通过 d 进行需要的格式化。

| 格式    | 示例       | 输出      | 描述                     |
| ------- | ---------- | --------- | ------------------------ |
| {:.2f}  | 3.1415926  | 3.14      | 保留小数点后2位          |
| {:.0f}  | 3.1415926  | 3         | 保留整数                 |
| {:0>2d} | 5          | 05        | 位数为2，数字补0，左补充 |
| {:x<4d} | 5          | 5xxx      | 位数为4，数字补x，右补充 |
| {:,}    | 1000000    | 1,000,000 | 以逗号分隔的数字格式     |
| {:.2%}  | 0.25       | 25%       | 百分比格式               |
| {:.2e}  | 1000000000 | 1.00E+0.9 | 指数记法                 |

```python
print("{:.2f}".format(3.1415926))
print("{:.0f}".format(3.1415926))
print("{:0>2d}".format(5))
print("{:x<4d}".format(5))
print("{:,}".format(1000000))
print("{:.2%}".format(0.25))
print("{:.2e}".format(1000000000))
```

>执行结果：
>
>3.14
>3
>05
>5xxx
>1,000,000
>25.00%
>1.00e+09
