函数

#### 1 调用函数

##### 1.1 abs函数

`abs`：绝对值

```python
>>> abs(100)
100
>>> abs(-1)
1
```



##### 1.2 max函数

`max`：返回最大值

```python
>>> max(2,3,1,-4)
3
```



##### 1.2 int函数，float函数，str函数，bool函数

```python
>>> int('123')
123
>>> int(12.34)
12
>>> float('11.23')
11.23
>>> float('1.23')
1.23
>>> str(100)
'100'
>>> str(1.23)
'1.23'
>>> bool(1)
True
>>> bool()
False
```



#### 2 自定义函数

定义一个函数要使用`def`语句

```python
# -*- coding: utf-8 -*-
#自定义一个求绝对值的my_abs函数
def my_abs(x):
    if x>=0:
        return x
    else:
        return -x
print(my_abs(-99))
```
