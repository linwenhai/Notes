## 函数

#### 11 函数len()

```python
# len()获悉列表的长度
cars = ['bmw', 'audi', 'toyota', 'subaru']
print(len(cars))
```

#### 13 函数range()

```python
# 函数range()从你第一个值开始数,并在到达第二个值后停，不包含第二个值
for value in range(1,5):
    print(value)
```

#### 14 列表统计计算

```python
x = [1,2,3,4,5,6,7,8,9]
print("min: ",min(x))

print("sum:",sum(x))
```
