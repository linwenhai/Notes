## 字典dictionary

字典是一系列键—值对，使用花括号{}。

```python
alien = {'color': 'green', 'points': 5}
```



#### 1 根据健访问字典中的值

```python
alien = {'color': 'green', 'points': 5}
print(alien['color'])
print(alien['points'])
```

```
输出结果：
green
5
```



#### 2 添加键—值对

```python
alien = {'color': 'green', 'points': 5}
alien['x_position'] = 2
alien['y_position'] = 7
print(alien)
```

```
输出结果：
{'color': 'green', 'points': 5, 'x_position': 2, 'y_position': 7}
```



#### 3 修改字典中的值

```python
alien = {'color': 'green', 'points': 5, 'x_position': 2, 'y_position': 7}
alien['color'] = 'yellow'
print(alien)
```

```
输出结果：
{'color': 'yellow', 'points': 5, 'x_position': 2, 'y_position': 7}
```
