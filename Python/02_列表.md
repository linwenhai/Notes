## 列表

`列表`由一系列按特定顺序排列的元素组成，用方括号[]表示列表，用逗号分隔元素。



#### 1 访问列表元素

```python
# 指定索引，访问列表元素
bicycles = ['trek', 'cannondale', 'redline', 'specialized']
print(bicycles[0])
```

>输出结果：
>
>trek



在Python中，第一个列表元素的索引为0，而不是1，最后一个元素索引为-1，倒数第二个元素索引为-2。

```python
bicycles = ['trek', 'cannondale', 'redline', 'specialized']
print(bicycles[0])
print(bicycles[1])
print(bicycles[-1])
print(bicycles[-2])
```

>输出结果：
>
>trek
>cannondale
>specialized
>redline



#### 2 修改列表元素

```python
# 指定列表名和元素索引
motorcycles = ['honda', 'yamaha', 'suzuki']
print(motorcycles)
motorcycles[0] = 'ducati'
print(motorcycles)
```

>输出结果：
>
>['honda', 'yamaha', 'suzuki']
>['ducati', 'yamaha', 'suzuki']



#### 3 在列表中添加元素

方法`append()`：将元素添加到了列表末尾。

```python
motorcycles = []
motorcycles.append('honda')
motorcycles.append('yamaha')
motorcycles.append('suzuki')
print(motorcycles)
```

>输出结果：
>
>['honda', 'yamaha', 'suzuki']



#### 4 在列表中插入元素

方法`insert()`：指定索引位置和新元素，添加到列表。

```python
motorcycles = ['honda', 'yamaha', 'suzuki']
motorcycles.insert(0, 'ducati')
print(motorcycles)
```

>输出结果：
>
>['ducati', 'honda', 'yamaha', 'suzuki']



#### 5 从列表中删除元素

方法`del`：指定索引删除元素。

```python
motorcycles = ['honda', 'yamaha', 'suzuki']
print(motorcycles)
del motorcycles[0]
print(motorcycles)
```

>输出结果：
>
>['honda', 'yamaha', 'suzuki']
>['yamaha', 'suzuki']



方法`pop()`：删除列表末尾的元素，并能够接着使用。

```python
motorcycles = ['honda', 'yamaha', 'suzuki']
print(motorcycles)
popped_motorcycle = motorcycles.pop()
print(motorcycles)
print(popped_motorcycle)
```

>输出结果：
>
>['honda', 'yamaha', 'suzuki']
>['honda', 'yamaha']
>suzuki



方法`remove()`：根据元素值删除元素（如列表有多个值，需使用for循环删除）。

```python
motorcycles = ['honda', 'yamaha', 'suzuki', 'ducati']
print(motorcycles)
motorcycles.remove('ducati')
print(motorcycles)
```

>输出结果：
>
>['honda', 'yamaha', 'suzuki', 'ducati']
>['honda', 'yamaha', 'suzuki']



#### 6 列表排序

方法`sort()`：对列表进行永久性排序。

```python
cars = ['bmw', 'audi', 'toyota', 'subaru']
# 首字母升序排列
cars.sort()
print(cars)
# 首字母降序排列
cars.sort(reverse=True)
print(cars)
```

>输出结果：
>
>['audi', 'bmw', 'subaru', 'toyota']
>['toyota', 'subaru', 'bmw', 'audi']



函数`sorted()`：对列表进行临时排序。

```python
cars = ['bmw', 'audi', 'toyota', 'subaru']
print(cars)
# 首字母升序排列
print(sorted(cars))
# 首字母降序排列
print(sorted(cars,reverse=True))
print(cars)
```

>输出结果：
>['bmw', 'audi', 'toyota', 'subaru']
>['audi', 'bmw', 'subaru', 'toyota']
>['toyota', 'subaru', 'bmw', 'audi']
>['bmw', 'audi', 'toyota', 'subaru']



方法`reverse()`：反转列表元素的排列顺序（非首字母排列）。

```python
cars = ['bmw', 'audi', 'toyota', 'subaru']
print(cars)
cars.reverse()
print(cars)
```

>输出结果：
>
>['bmw', 'audi', 'toyota', 'subaru']
>['subaru', 'toyota', 'audi', 'bmw']



#### 7 遍历整个列表

```python
# 使用for循环遍历整个列表
magicians = ['alice', 'david', 'carolina']
for magician in magicians:
	print(magician)
```

> 输出结果：
>
> alice
> david
> carolina



#### 8 创建数字列表

函数`range()`：生成一系列的数字。

```python
for value in range(1,5):
	print(value)
```

> 输出结果：
>
> 1
> 2
> 3
> 4



函数`list()`：将`range()`的结果直接转换为列表。

```python
numbers = list(range(1,6))
print(numbers)
```

> 输出结果：
>
> [1, 2, 3, 4, 5]



```python
# 创建一个列表，其中包含前10个整数（即1~10）的平方
squares = []
for value in range(1,11):
	squares.append(value**2)
	print(squares)
```

> 输出结果：
>
> [1, 4, 9, 16, 25, 36, 49, 64, 81, 100]



`列表解析`：

```python
# for后面无引号
squares = [ value**2 for value in range(1,11)]
print(squares)
```

> 输出结果：
>
> [1, 4, 9, 16, 25, 36, 49, 64, 81, 100]









#### 9 列表统计计算

```python
>>> digits = [1, 2, 3, 4, 5, 6, 7, 8, 9, 0]
>>> min(digits)
0
>>> max(digits)

```



#### 10 切片

```python
# 使用切片提取列表元素
players = ['charles', 'martina', 'michael', 'florence', 'eli']
print(players[0:3])
print(players[1:4])
```

> 输出结果：
>
> ['charles', 'martina', 'michael']
> ['martina', 'michael', 'florence']



```python
# 如没有指定第一个索引，将自动从列表开头开始
print(players[:4])
```

> 输出结果：
>
> ['charles', 'martina', 'michael', 'florence']



```python
# 如没有指定最后索引，将提取到列表末尾的所有元素
players = ['charles', 'martina', 'michael', 'florence', 'eli']
print(players[2:])
```

> 输出结果：
>
> ['michael', 'florence', 'eli']



```python
# 使用for循环遍历切片
players = ['charles', 'martina', 'michael', 'florence', 'eli']
for player in players[:3]:
	print(player.title())
```

> 输出结果：
>
> Charles
> Martina
> Michael



#### 11 复制列表

要复制列表，可创建一个包含整个列表的切片，方法是同时省略起始索引和终止索引（[:]）。

```python
my_foods = ['pizza', 'falafel', 'carrot cake']
friend_foods = my_foods[:]
```



#### 12 元组

不可变的列表被称为元组，元组使用圆括号表示，语法和列表类似。

```python
dimensions = (200, 50)
print(dimensions[0])
print(dimensions[1])
```

> 输出结果：
>
> 200
> 50
