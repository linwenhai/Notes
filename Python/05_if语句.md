## if语句

### 1 if-else 语句

```python
age = 12
if age < 4:
    price = 0
else:
    price = 10
print("Your admission cost is $" + str(price) + ".")
```



### 2 if-elif-else 语句

```python
age = 12
if age < 4:
    price = 0
elif age < 18:
    price = 5
else:
    price = 10
print("Your admission cost is $" + str(price) + ".")
```
