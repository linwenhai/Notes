## Haproxy配置详解

haproxy 的配置文件由两部分组成：全局设定和对代理的设定，共分为五段：global，defaults，frontend，backend，listen。



### 1 global



| 参数    | 描述                              |
| ------- | --------------------------------- |
| chroot  | haproxy的工作目录                 |
| daemon  | haproxy以守护进程的方式工作于后台 |
| group   | 组名                              |
| user    | 用户名                            |
| maxconn |                                   |
