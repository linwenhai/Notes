## 正则表达式



| 字符   | 描述                                |
| ------ | ----------------------------------- |
| ^      | 匹配输入字符串的开始位置            |
| $      | 匹配输入字符串的结束位置            |
| [0-9]  | 匹配单个数字                        |
| [0-9]+ | 匹配多个数字                        |
| {3,15} | 3~15个字符长度                      |
|        |                                     |
| [ABC]  | 匹配 **[...]** 中的所有字符         |
| [^ABC] | 匹配除了 **[...]** 中字符的所有字符 |
| [A-Z]  | 匹配所有大写字母                    |
| [a-z]  | 匹配所有小写字母                    |
