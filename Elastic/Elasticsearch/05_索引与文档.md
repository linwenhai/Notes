## 索引与文档



### 1 索引HTTP请求方法

```bash
PUT test	# 创建索引

GET test	# 查看索引

HEAD test	# 检验索引

DELETE test	# 删除索引
```



### 2 索引别名aliases

索引别名类似关系数据库中的视图。

```json
PUT students
{
  "aliases": {
    "girls": {
      "filter": {
        "term": {
          "gender": "F"
        }
      },
      "routing": "F"
    },
    "boys":{
      "filter": {
        "term": {
          "gender": "M"
        },
        "routing":"M"
      }
    }
  }
}
```
