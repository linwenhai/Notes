## 分析器（analysis）

分析器（analysis）组成：

- 字符过滤器（Character Filter）
- 分词器（Tokenizer）
- 分词过滤器（Token Filter）





### 1 _analyze接口

```json
# 使用standard分析器处理文本
POST _analyze
{
  "analyzer": "standard",
  "text": "I live duoduo"
}
```
